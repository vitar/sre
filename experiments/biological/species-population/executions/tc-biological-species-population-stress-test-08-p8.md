# Execution 08 — Problem 8: Level Discipline vs Organism Aggregation

## Scope
Problem tested: **Can population evidence support TC without hiding organism-level assumptions?**

## Decision
Yes, if emergent population structure and distributional effects are analyzed directly, not only individual averages.

## Case ID
BIO-2-008

## Biological context
Metapopulation where patch occupancy and dispersal structure drive outcomes.

## Unit boundary
Metapopulation network.

## Time horizon
10–12 generations.

## Environment
Uneven patch quality, corridor disruption, localized shocks.

## Resource constraints
Patch carrying capacity, dispersal routes, mate encounter rates.

## Observed demands
Patch persistence, reproduction across network, adaptive redistribution.

## Proposed TC contour mapping
- Survival: persistence of metapopulation occupancy.
- Reproduction: successful recruitment across patches.
- Evolution: network-level strategy shift improving future resilience.

## Structural elements mapping
Code (metapopulation response repertoire), Boundary (patch network membership), Gate (corridor permeability), Signal (patch quality indicators), Receiver (subpopulations with distinct responsiveness).

## Mediators
Connectivity architecture, density dependence, local disturbance regime.

## What supports TC
- Network structure creates outcomes not inferable from average individual strategy alone.
- Level-separated analysis resolves contradictory readings.

## What challenges TC
- Aggregation can hide local contour differences and falsely simplify diagnosis.

## Confidence
High

## Verdict
**Supports** with strict level-separation discipline.
